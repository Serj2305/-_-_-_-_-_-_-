Главная

Топографические (картографические) условные знаки — символические штриховые и фоновые условные обозначения объектов местности, применяемые для их изображения на топографических картах. Для топографических условных знаков предусмотрена общность обозначений (по начертанию и цвету) однородных групп объектов, при этом основные знаки для топографических карт разных стран не имеют между собой особых различий.
